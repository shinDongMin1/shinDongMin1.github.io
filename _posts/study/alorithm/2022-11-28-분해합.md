## 코드
```cpp
#include<iostream>
using namespace std;

int main() {
    cin.sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);

    string s;
    bool flag;
    while (true) {
        cin >> s;
        flag = true;
        if (s[0] == '0') break;

        for (int i = 0; i < (s.size() / 2); i++) {
            if (s[i] != s[s.size() - 1 - i]) {
                flag = false;
                break;
            }
        }
        if(flag == true) 
            cout << "yes\n";
        else
            cout << "no\n";
    }
}
```
## 설명
기본적으로 C/C++의 기본 입출력함수를 혼합하여 사용되게 만들었지만 그만큼 실행속도에서 차이가 생기기 때문에 설정을 해주며 입력을 string으로 받아 양쪽 끝을 시작으로 비교하며 만약 다른 수가 나올때 flag 값을 변경해주고 경우에 맞게 출력합니다.
