## 코드
```Python
import sys
n = int(input())
arr = [ 0, 1, 2]

for i in range(3, n + 1):
    arr.append(arr[i-2] + arr[i-1])

print(arr[n] % 10007)
```
## 설명
파이썬을 통해서 사용자로부터 입력받아 자료형 list을 사용하여 dp알고리즘으로 2xn 타일링를 구현했습니다. <br>
